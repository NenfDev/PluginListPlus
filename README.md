A clean, styled replacement for /plugins, /pl, and /plist.



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



◆ 5 Display Styles   ◆ 10 Themes   ◆ GUI Config   ◆ Book View   ◆ Private Mode



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



Replaces the default plugin list with a cleaner, styled, customizable version for those that also are not fond of the default list.





▌ DISPLAY STYLES

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



COMPACT ⟫ Fits everything into as few lines as possible.

GROUPED ⟫ Splits enabled and disabled plugins for fast status checking.

BOXED ⟫ One plugin per line with a clean framed layout.

CATEGORIZED ⟫ Sorts plugins by Paper, Bukkit, and status.

DETAILED ⟫ Shows versions, authors, badges, and extra plugin info.





▌ THEMES

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



Gold   Crimson   Emerald   Sunset   Ocean

Galaxy   Cherry   Arctic   Neon   Rainbow





▌ ADMIN TOOLS

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



Interactive Config Panel ⟫ Change style, theme, sorting, badges, versions, legend, and gradients in-game.

Inventory GUI ⟫ Configure the plugin visually without typing commands manually.

Book View ⟫ Open the plugin list in a paginated written book format.

Inventory Browser ⟫ Browse plugins one at a time in a clean paginated GUI.

Private Mode ⟫ Hide plugin info from players and show a custom privacy message instead.

Startup Error Lookup ⟫ Check the last startup error for any disabled plugin directly in-game.





▌ COMMANDS

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



/pluginlist ⟫ Shows the saved layout.

/pluginlist config ⟫ Opens the interactive config panel.

/pluginlist gui ⟫ Opens the GUI setup menu.

/pluginlist browse ⟫ Opens the paginated inventory browser.

/pluginlist book ⟫ Opens the written book view.

/pluginlist error [plugin] ⟫ Shows the last startup error for that plugin.





▌ INSTALLATION

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━



1. Place the jar into your plugins folder.

2. Restart your server.

3. Run /pluginlist and pick your style.
